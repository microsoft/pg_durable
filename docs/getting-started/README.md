# pg_durable Getting Started Website

This folder contains a static, scenario-driven landing page for pg_durable users.

## Files

- `index.html` — Main page
- `styles.css` — Page styling

## Preview locally

From the repository root:

```bash
python3 -m http.server 8080
```

Then open:

- <http://localhost:8080/docs/getting-started/index.html>

## Content sources

The website content is based on:

- `docs/SCENARIOS.md`
- `docs/ai/SCENARIOS.md`
- `USER_GUIDE.md`
- `README.md`
